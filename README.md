To run the game, In you browser enter: 
https://LcndrA.github.io/PlinkoDemo


Hub Controls:
W - Forward 
A - Left
S - Backwards
D - Right
E - Interact
Spacebar - Jump
Mouse - Camera controls

Board Controls:
W - Up (Player 1)
A - Left (Player 1)
D - Right (Player 1)
Spacebar - Drop ball (Player 1)
Up Arrow - Push Ball Away (Player 2)
Mouse - controls Magnet
